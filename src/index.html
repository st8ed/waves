Export as text

Для сохранения в режиме совместимости со старыми браузерами выполните следующую последовательность действий:

  • Выделите весь текст ниже (Ctrl+A)
  • Скопируйте его в буфер обмена (Ctrl+C)
  • Сохраните вставкой (Ctrl+V) в виде текстового документа с помощью приложения "Блокнот (Notepad)" или иного элементарного текстового редактора.
  • Расширение файла рекомендуется выбрать json, кодировка - UTF-8
  • Microsoft Word добавляет в файл свои мета-данные и поэтому не может быть использован для этой цели.
